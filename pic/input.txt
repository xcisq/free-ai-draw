## Overall Summary
This work addresses the critical challenge of creating high-quality methodology charts, such as flowcharts, system architecture diagrams, and experimental pipelines, which are essential for conveying research logic and enhancing paper readability. Despite their importance, researchers currently face a dilemma of high investment and low efficiency. Traditional tools like Visio and PowerPoint require cumbersome manual operations for element arrangement and style unification, while modern online platforms like Draw.io and Excalidraw remain confined to a manual drag-and-drop paradigm lacking semantic understanding. Although Large Language Models and multimodal models offer new possibilities for natural language-driven drawing, existing approaches suffer from significant prompt bottlenecks and deficiencies in spatial relationship accuracy and structure recoverability, as evidenced by evaluations like ScImage and SciFlow-Bench. To overcome these limitations, this research proposes an integrated scientific drawing system that combines intelligent prompt generation, AI agent collaboration, and human interactive refinement. Built upon the open-source whiteboard tool Drawnix and the Plait drawing framework, the system aims to achieve an end-to-end intelligent process from natural language requirements to editable vector charts. The core contribution lies in lowering the threshold for AI drawing, covering the full link of intent parsing to style transfer through multi-agent collaboration, and ensuring zero-loss rendering via native JSON formats.

## Complete Method Description
The proposed methodology centers on constructing a unified system that bridges the gap between natural language intent and structured, editable scientific visuals. The technical foundation relies on the open-source whiteboard tool Drawnix, which is based on the Plait drawing framework. This choice of platform is strategic, as it enables the system to leverage Plait JSON native formats. This technical specification is crucial for achieving the goal of zero-loss rendering and full attribute editability, ensuring that the generated charts are not merely static images but structured objects that researchers can modify without quality degradation. The method is designed to resolve the specific failures of current generative models, particularly their inability to maintain spatial accuracy and structured editability in academic scenarios.

The operational logic of the system is driven by three interconnected core objectives that form the narrative of the method. First, the system addresses the prompt bottleneck by implementing intelligent prompt generation. This component is designed to reduce the burden on users who previously needed to repeatedly debug prompts to achieve expected outputs when using general models like GPT-4 or Claude. By automating the refinement of natural language inputs, the system lowers the technical barrier for researchers, allowing them to focus on scientific content rather than engineering the interaction with the AI.

Second, the method employs a multi-agent collaboration mechanism to cover the complete workflow of chart creation. Instead of relying on a single model pass, the system distributes tasks across specialized agents responsible for intent parsing, layout optimization, and style transfer. This division of labor is intended to enhance the structural integrity of the output. The intent parsing agent ensures the semantic requirements are understood, the layout optimization agent addresses the spatial relationship accuracy issues highlighted by prior benchmarks, and the style transfer agent ensures the visual output meets academic standards. This collaborative approach aims to overcome the limitations where single-model outputs lack the necessary structure for direct academic application.

Third, the method integrates human interaction refinement into the generation loop. Recognizing that fully automated generation may not always meet precise requirements, the system facilitates a human-in-the-loop process where users can edit the generated vector charts. Because the output is maintained in Plait JSON native format, this refinement process is seamless and does not incur rendering losses. This combines the efficiency of AI generation with the precision of manual adjustment. The research significance is framed through this methodological integration, offering a new paradigm for the deep fusion of large language models and interactive drawing tools. The engineering implementation directly contributes to the open-source community via Drawnix, providing a reusable technical scheme for enhancing AI capabilities in similar tools, while the academic contribution lies in providing a systematic methodology for the emerging direction of natural language to structured charts.
