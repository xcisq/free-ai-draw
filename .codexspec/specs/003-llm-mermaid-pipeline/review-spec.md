## 详细发现

### 关键问题（必须修复）

无关键问题。

### 警告（应该修复）

- **[SPEC-001]**：REQ-003 信息收集结构表格中，"预设选项"没有列出具体选项内容
  - **影响**：开发时可能需要回头澄清具体选项
  - **建议**：补充各信息类别的预设选项清单，例如：
    - 布局方向：从左到右 / 从上到下 / 自适应
    - 样式偏好：专业风格 / 活泼风格 / 学术风格 / 极简风格
    - 节点形状：圆角矩形 / 矩形 / 菱形（暂不需要）/ 圆形

- **[SPEC-002]**：REQ-007 中环境变量前缀不一致
  - **影响**：可能与其他功能的环境变量混淆
  - **建议**：使用统一前缀 `VITE_LLM_MERMAID_*` 而非 `VITE_LLM_*`，与 PaperDraw 的 `VITE_PAPERDRAW_*` 保持一致

- **[SPEC-003]**：NFR-002 "可扩展性"缺少具体指导
  - **影响**：扩展性设计可能不够明确
  - **建议**：补充具体的设计指引，例如"对话系统使用策略模式，新图表类型通过注册机制添加"

### 建议（可选）

- **[SPEC-004]**：建议在"输出示例"章节增加"错误输出示例"
  - **价值**：帮助开发者理解错误处理的预期行为

- **[SPEC-005]**：TC-003 样式调整测试用例中，"虚线边框"的实现方式需要明确
  - **价值**：确保 Mermaid classDef 支持虚线边框（实际上可能需要用 `stroke-dasharray`）

- **[SPEC-006]**：建议增加"撤销/重做"相关的需求说明
  - **价值**：用户在调整样式时可能需要回退操作
